RMK-TransitGo is a real-time campus transit platform built to eliminate the inefficiencies of manual bus management. It integrates live GPS tracking with a proximity-gated Wait for Me alert, allowing students to request departure holds only when within a verified range of their bus — keeping requests relevant and non-disruptive.
Drivers receive a clear, structured interface for trip management and student alerts, while administrators get a secure portal for user verification, fleet control, and trip analytics. Built on React Native and Supabase, the system enforces role-based access, live data sync, and cross-platform compatibility — delivering a reliable, well-coordinated commute experience for the entire college community.
